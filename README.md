# Rotation College Email Verification

This is the web verification page for Rotation's college email verification system.

## Setup

1. This repository is configured for GitHub Pages
2. The verification page handles college email verification links from emails
3. Users click verification links → get verified → redirected to mobile app

## Files

- `index.html` - Redirects to the verification page
- `verification_redirect.html` - Handles the actual verification process
- `README.md` - This file

## Deployment

This is automatically deployed via GitHub Pages when pushed to the main branch.
